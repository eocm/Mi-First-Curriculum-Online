    Sitio Web de una Biblioteca

    BLIM

    Juego de Automata

En este proyecto se creo un juego de dados en donde se tenian distintas reglas
para ir ganando puntos, si caias en una regla de esas podias ganas o restar
puntos en un determinado numero de tiros validando las reglas.
Para este Proyecto se utilizo Unity, C#.
